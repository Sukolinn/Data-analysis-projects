# Дополнительные материалы

В исходном архиве кейса здесь лежали большие справочные PDF по флотации, металлургии и обогащению.

Они не включены в GitHub-версию, потому что часть файлов слишком большая для обычной загрузки в репозиторий. Для финальной сдачи их лучше положить в облачный архив вместе с ZIP проекта, видео и презентацией.

Рабочий прототип использует компактные данные из `01_Data/processed`, XLSX/DOCX по фабрикам, PNG-схемы и регламенты.
