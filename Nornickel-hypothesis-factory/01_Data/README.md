# 01_Data

Раздел с входными данными проекта.

Содержимое:

- `processed/nornickel_real_case_data_summary.json` — подготовленная структурированная сводка по XLSX-отчетам.
- `processed/nornickel_real_case_knowledge_base.md` — компактная база знаний реального кейса, используемая приложением.
- `demo/` — небольшой резервный демо-набор.
- `raw/Задача 1. Фабрика гипотез/` — исходные XLSX, DOCX, PNG-схемы и регламенты.

Большие справочные PDF из дополнительных материалов лучше хранить в облачном архиве, а не в GitHub.
