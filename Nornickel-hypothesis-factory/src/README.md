# Исходный код

В этой папке находится код интеллектуальной системы генерации гипотез.

## Структура

- `backend/` — бэкенд: обработка данных, RAG-пайплайн, работа с LLM, анализ данных.
- `frontend/` — интерфейс на Streamlit.
- `requirements.txt` — зависимости проекта.

## Запуск

1. Установить зависимости:
`pip install -r requirements.`
2. Запустить бэкенд (из папки backend/):
`python app.py`
3. Запустить фронтенд (из папки frontend/):
`streamlit run app.py`

## Примечание

Детали реализации добавляются разработчиками по мере написания кода. Требования к функциональности зафиксированы в корневом README.md.
